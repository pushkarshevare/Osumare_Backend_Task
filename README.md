Task API
1. Clone the repository:

    ```
    git clone https://github.com/your-username/task-api.git
    cd task-api
    ```

2. Install the dependencies:

    ```sh
    npm install
    ```
3. Start the server:

```sh
node src/index.js
```

Get all Todo List
URL: '/todos'
Method: GET

Create a new todo
URL: '/todos'
Method: POST

Get a todo by ID
URL: '/todos/:id'
Method: GET

Update an existing todo
URL: '/todos/:id'
Method: PUT

Delete a todo by ID
URL: '/todos/:id'
Method: DELETE

Note : You can use Postman for API Testing.
