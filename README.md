Task API
1. Clone the repository:

    ```
    git clone https://github.com/pushkarshevare/Osumare_Backend_Task.git
    ```

2. Install the dependencies:

    ```sh
    npm install
    npm i nodemon
    ```
3. Start the server:

```sh
nodemon main.js
```

Get all Todo List :  
URL: '/todos', 
Method: GET

Create a new todo :  
URL: '/todos',
Method: POST

Get a todo by ID :  
URL: '/todos/:id',
Method: GET

Update an existing todo :  
URL: '/todos/:id',
Method: PUT

Delete a todo by ID :  
URL: '/todos/:id',
Method: DELETE

Note : You can use Postman for API Testing.
